Provides a low-level interface to the encoding/hex package.

Package hex implements hexadecimal encoding and decoding.

  • DecodeString

    Function v1.0 
    (DecodeString __s)

    DecodeString returns the bytes represented by the hexadecimal string s.

    DecodeString expects that src contains only hexadecimal
    characters and that src has even length.
    If the input is malformed, DecodeString returns
    the bytes decoded before the error.

    Go input arguments: (s string)

    Go return type: ([]byte, error)

    Joker input arguments: [^String s]

    Joker return type: [(vector-of Int) Error]

  • DecodedLen

    Function v1.0 
    (DecodedLen __x)

    DecodedLen returns the length of a decoding of x source bytes.
    Specifically, it returns x / 2.

    Go input arguments: (x int)

    Go return type: int

    Joker input arguments: [^Int x]

    Joker return type: Int

  • EncodedLen

    Function v1.0 
    (EncodedLen __n)

    EncodedLen returns the length of an encoding of n source bytes.
    Specifically, it returns n * 2.

    Go input arguments: (n int)

    Go return type: int

    Joker input arguments: [^Int n]

    Joker return type: Int

  • ErrLength

    GoVar[*.error] v1.0 
    
  
    ErrLength reports an attempt to decode an odd-length input
    
using Decode or DecodeString.
    
The stream-based Decoder returns io.ErrUnexpectedEOF instead of ErrLength.
